lookup-eqn-string

lookup-eqn-string eqn-string &optional id

Description
Check correctness of a student equation entry.
Example
  "(lookup-eqn-string \"F = ma\" 0)"
Returns

An Entry Status Return Value .
 
Arguments
eqn-string:
the string entered by the student
id:
the integer id of the equation box used (0-based array)
