delete-equation

delete-equation id
Description
Deletes a student equation from the student-entries list.
Returns
The new student-equations list. Return value is ignored by the workbench.
Arguments
id:
The id of the equation being deleted. Integer id starts at 0 for first equation box.
Side effects
Removes this student entry from the entered-by fields of each of the corresponding system equations.
