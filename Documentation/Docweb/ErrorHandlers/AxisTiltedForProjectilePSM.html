Axis tilted for projectile PSM

(as of 3/27/01)
If the student's entry is an axis
  • that does not have a zero rotation
  • and making it have zero rotation makes it correct
  • and the axis is part of a projectile motion PSM
then the hint sequence is
  • run the preamble up to the projectile PSM
  • hint/teach/state that whenever one is using the PSM method, the axes should be in standard (zero) rotation.
