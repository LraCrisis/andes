Body slip

(as of 3/27/01)

This is a default error handler.  It must be applied only if more specific cases don't handle the error.

If the student's entry is not an equation

  • and it contains a body entry
  • and changing the body entry to ?body makes it correct
then the hint sequence is
  • (hint string (point) "Check your body.")
  • (hint string (bottom-out) "You need to specify ~a as the body." ?body)
