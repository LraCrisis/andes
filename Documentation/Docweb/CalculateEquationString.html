calculate-equation-string

calculate-equation-string string new-id
Description
Takes an equation, substitutes all known values that have been entered by the student into it and then simplifies the result. Returns the simplified string. Lookup-eqn-string will be called on the original equation before calling this function to update the correct/incorrect status. Substitutions will only be done for *correct* values that the student has entered. Therefore the correct/incorrect status of the resulting equation will be the same as the original equation (ie. if the equation being substituted into is correct, the result will be correct as well.
Returns
The string representing the simplified equation if it could be simplified, otherwise nil
Arguments
string:
the equation string to be substituted into
new-id:
the id of the new equation to be entered
Side Effects
Creates a new equation entry representing the newly simplified equation.
