Makefile for the Andes2 algebra system

The Makefile is set up assuming the there are three sister directories, src, gpp and run, each daughter of the same directory, which in Joel's project is c:/joel/math/algebra. The programs which use files themselves assume the input equation files are in c:/joel/math/data/eqs.wur.

The Makefile assumes all the source code is in src, the Makefile and the .o files are to reside in gpp, and the executable programs are to be placed in run. The Makefile below has long historical roots, with obsolete programs included. It is only the two programs, colander and allprog, which are given by make without arguments, that are still being used and maintained. The Solver.dll cannot be made with this Makefile, but needs to be made with VisualC++.
