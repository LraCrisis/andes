Classes defined in the Andes2 algebra system

The classes defined in the system are
  • dimensThe SI physical units for a variable or expression.
  • expr An algebraic expression, possibly an equation. This is a virtual class with subclasses
    • primexpr a virtual subclass with numvalexp and physvarptr and fakeexpr as subclasses
    • fakeexpr A subterfuge of use only during parsing
    • numvalexp A numerical value, with physical units, such as 2.5 m/s
    • physvarptr A variable, represented by a pointer to a physvar
    • binopexp A binary operator between expr's. The available operators are divided-by, raised-to-power, equals, and the unimplemented greater-or-equal and greater-than.
    • functexp A known function of a single argument. The available functions are sin, cos, tan, exp, ln, log10, sqrt, abs
    • n_opexp An n-ary operator among expr's. The available operators are plus and mult.
  • physvarA physical variable. It contains information on attributes, units, names, etc.
  • indyset A set of independent functions and methods for determining whether the linear expansion of a new function is independent of those in the set, and if not, giving its expansion.
  • valander A combination of the value and gradient of a function at the solution point of the problem, and information about whether the function depends on each variable.
  • answitherr: A combination of the numerical value and absolute error in a numerical expression.
  • unitabrs: A list of abbreviations for units, unit prefixes, and preferred ways of expressing combinations of units.
  • oper: a class with a fixed set of instantiations, one for each function type for a functexp or each operator for a binopexp or n_opexp.
