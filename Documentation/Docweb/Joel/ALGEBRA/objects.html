Description of global objects used by the Andes algebra system

There are a number of global objects defined by the algebra system. Most, but not all, are defined in extstruct.h and instantiated in whichever .cpp file declares itself IAmMain. The global objects are
vector<physvar *> * canonvarslist of the canonical variables
vector<binopexp *> * canoneqflist of the canonical equations
vector<binopexp *> * paramasgnlist of parameter assignment statements
vector<binopexp *>  studeqflist of the student equations, by slot
vector<string *>  studeqsoriglist of the student equations as presented. Not very useful if the algebra system doesn't get the originals.
vector<double> * numsolsThe numerical value (in SI units) of the canonical variables in the solution.
vector<string> * constnameslist of physical constants, including Pi
vector<numvalexp *> * constnumvals   list of the values of the physical constants.
vector<valander *> * canongrads;gradients of the canonical equations
vector<valander *>  studgradsgradients of the student equations
vector<indyset> * listofsets;list of independence sets
vector<vector<int> > * listsetrefs;list of lists, one for each independence set, of the indices in canoneqf of the equations in the set.
vector<int> * lasttriedeq;index in canoneqf of the last equation isindiy'd for this set
int numindysets;The number of independence sets currently.
int numvars;The number of variables for this problem.
bool setupdonetrue once initial setup is done. Never falsified.
numparamsThe number of parameters in this problem.
bool gotthevarsset by indyDoneAddVariables when all variables have been declared
unittabletable of unit abbreviations and prefixes. Once made during setup it remains unchanged.

The structures below are declared in extstruct.h but are never used. They were intended to hold the student variables before the decision was made to have the help system do the parsing of student equations and translation to canonical variables.
vector<string*>  studvars list of declared student variables
vector<expr*>  studvarvalslist of the values of those variables in the solution
vector<physvar*>  studvarcanon   expression in canonical variables for the student variable.
vector<binopexp*> * solsexprlist of the solutions as symbolic assignment statements
