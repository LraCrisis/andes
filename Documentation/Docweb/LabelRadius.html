label-radius

label-radius label id name
Description
Label the radius of revolution of an object moving in a circle.
Example
  "(label-radius \"r\" 'radius-21 'car)"
Returns
An Entry Status Return Value.
Arguments
label
the label given to the radius by the student, a string
id
the id of the radius object
name
the body that is moving in a circle with this radius
Side Effects
Add a variable for the radius to the list of variables.
