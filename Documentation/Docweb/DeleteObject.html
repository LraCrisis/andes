delete-object

delete-object label &optional id
Example
  "(delete-object \"F\" 'vector-1)"
Description
Delete the student defined object with the given label and/or id from the student-entries list.
Returns
The new student-entries list. Return value is ignored by the workbench.
Arguments
label:
the label given to the object by the student
id:
the unique id assigned to the object by the workbench
Side effects
Marks the corresponding system entries as "unentered."  If the object involves a variable, then removes that variable from the symbol table.
