why-wrong-object

why-wrong-object &optional label id
Description
Get a message explaining why an entry was judged to be incorrect. The message is displayed in the workbench hint window. The messages for why-wrong-object are created when the object is first entered, so all this function has to do is find the string and return it.
Example
  "(why-wrong-object \"F\" 'vector-1)"
Returns
A hint descriptor return value
Arguments
label:
The label (may not be unique) given to the object by the student.
id:
The unique id assigned to the object by the workbench
