check-answer

check-answer answer answer-id
Description
Lookup a student's answer in the answer box.
Example
 "(check-answer \"a=4.5\" 'ANSWER-1)"
 "(check-answer \"4.5 m/s\" 'ANSWER-1)"
Returns
An Entry Status Return Value
Arguments
answer:
the contents of the answer field the student entered.
answer-id:
the author-defined id for the answer field. Must start with the string "ANSWER"
