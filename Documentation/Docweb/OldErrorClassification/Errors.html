Examples of errors and how to handle them

(as of 3/23/01)

Non-equation errors

  • Errors that can be interpreted by matching
  • Errors that can't be interpreted
  • Andes1 errors that Andes2 should mark as correct

Equation errors

  • Errors that can be interpreted via perturbation of the student's equation
    • Insert negative signs
    • Replace sine by cosine, etc.
    • Replace student variable by a correct quantity
    • Replace student's value for a given quantity by its correct value
    • Put missing forces into a sum of forces
  • Errors that can be interpreted by literal matching
  • Uninterpretable equation errors
