hint-next-substep

hint-next-substep
Description
Triggered when the student clicks on "how do I do that?" Finds the next substep in the goal tree after the last hint and produces a hint about that substep.
Example
  "(hint-next-substep)"
Returns
A hint descriptor return value.
Arguments
none.
Side Effects
Updates the Assessor bayesian network to reflect the fact that the student has received this hint.
