explain-more
Description
Triggered when the student clicks on "explain further." Gives more specific information about the last hint that was given. Looks at the list of hint templates for the item in question and finds the next-most-specific one.
Example
  "(explain-more)"
Returns
A hint descriptor return value.
Arguments
none.
Side Effects
Updates the Assessor bayesian network to reflect the fact that the student has received this hint.
