why-wrong-equation

why-wrong-equation id
Description
Get a message explaining why the equation entry is not correct. Unlike with diagram objects, computing the what's wrong message is done when why-wrong-equation is called rather than when the equation is entered, since it can take a few seconds to complete.
Example
  "(why-wrong-equation 0)"
Returns
A hint descriptor return value
Arguments
id:
The number of the equation in the 0 based equation array.
