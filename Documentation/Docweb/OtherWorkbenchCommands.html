Other Workbench commands

The following list from the workbench source code lists other commands in the workbench command string interface. These commands are used when replaying log files and are not primarily intended for use by the help system. However, they could in principle be used to construct complex scripts used while giving help. A rudimentary demo playing facility was implemented in the workbench to construct demonstrations using a moving pointer on the screen by annotating Andes log files to insert the demo messages, for example. However, this facility was not used in ANDES since we lacked the resources to actually develop good demos.

Detailed documentation of the commands and their parameters is only available in the workbench source code. This list is just included to give an idea of what’s available. If you see something you’re interested in, just ask. The workbench uses a table-driven command dispatcher so further commands can easily be added to meet the needs of the help system.

// Generic dialog box operations:
F                    set focus to control
C                    change edit control text
c                    change rich edit control text
DRP                  drop combo box’s drop-down list
SEL                  select from combo box
CLOSE                close combo box’s drop-down list
SEL-LIST             select from list box
BTN-CLICK            click button
Mv-dlg               move the whole dialog box

// Equation View operations
EQ-F                 Set focus to an equation
K                    Kill focus on an equation
E                    change equation text
EQ-SUBMIT            submit equation to help system
EQ-DELETE            “delete” equation, i.e. change to blank
EQ-Whatswrong        Ask what’s wrong on an equation
EQ-Calculate         Ask to calculate an equation
EQ-RESULT            Receive checking result
EQ-SolveFor          Ask to solve for a variable

// Diagram view operations
L                    Press left mouse button
M                    Drag mouse to new position while pressed
Up                   Release mouse button
Select-tool          Select a drawing tool
Begin-draw           Begin drawing an object
Begin-vector         Begin drawing a vector
Begin-Resize         Begin resizing an object
Begin-move           Begin moving an object
Click-bg             Click background to clear selection
Select               Selected an object
Deselect             Deselected an object
Delete               Delete an object
Dup                  Duplicate an object
Menu                 Popped up context menu
Mark                 Marked a vector resolved
Edit-props           Open object’s dialog box
Zdir                 Set Z-Axis direction

// for answer boxes:
Ans-enter            Set focus to answer box
Ans-exit             Focus left answer box
Ans                  Changed answer box text
Ans-submit           Submit answer box to helpsys

// Help
FBD-Whatswrong       Asked what’s wrong on a drawn object

// Training card input messages
Tcard                Pop up training card

// Variable view operations
Select-Variable      Select given variable
New-Variable         Start defining new variable
Modify-Variable      Edit existing variable
Delete-Variable      Delete a variable
Variable-Whatswrong  Ask what’s wrong on a variable

// Application Operations
Open-Problem        Open a problem
Open-Example        Open an example
Close               Close problem
CLOSE-APP           Close whole application

// App Help request commands:
Help-Hint           Ask for a hint
Help-Explain        Ask for next hint
Help-How            Ask for next substep
Help-Why            Ask why
Textbook            Open physics textbook (experimental)
Comment             Submit a comment

// Hint pane operations
Hint-Hide          Hide the pane
Hint-PopupDef      Show a pop-up definition
Kill-Popup         Hide any pop-up definition
Discourse         (Atlas) type free text response

// for manipulating demo pointer agent in demo scripts:
Point-to          Move pointer to an object
Demo-msg          Show a message near the pointer
