lookup-vector

lookup-vector label avg-inst type system dir mag &optional time id
Description
Check correctness of a vector drawn by the student. May be any vector type except force.
Example
  "(lookup-vector \"a1\" 'average 'acceleration 'b1 270 nil '$|$T1 to T2$|$ nil 'vector-1)"
Returns
An Entry Status Return Value.
Arguments
label:
the vector label, a string
avg-inst:
is the vector an average or instantaneous quantity? Value is either 'average or 'instantaneous.
type:
the type of vector: velocity, acceleration, displacement, etc.
system:
the body that is moving. May be system label (student defined) or body name (given).
dir:
angle of the motion vector from horizontal: 0dir360 or a negative number coding a z-axis direction as follows
-1 => out of the plane; -2 => into the page; -3 =>unknown but along the z axis.
mag:
magnitude of the vector or NIL if unspecified
time:
the time period during which the vector is constant. If nil, and system is a student-defined system, the time will be taken from the system definition.
id:
id assigned to vector by workbench
Side Effects
If the vector is correct, the help system marks the corresponding system entry as "entered," defines the magnitude and direction variables, and enters the variables in the symbol table.
