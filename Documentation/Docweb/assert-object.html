assert-object, assert-compound-object

assert-object label name &optional time id
assert-compound-object label names &optional time id
Description
Checks correctness of a student defined single or compound body.
Example
  "(assert-object \"b1\" 'blk nil 'system-1)"
Returns
Entry status return value
Arguments
label:
the label of the body, a string
name(s):
the name the body(s) was given in the problem description
time:
the time period specified for the system
id:
id assigned to this object by the workbench
Side Effects
Marks the corresponding system entry as "entered."  Defines a mass variable whose name has "m" concatentated to the given label.  Enters into the symbol table this name paired with the system's name for the same quantity.
