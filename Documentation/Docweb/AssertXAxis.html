assert-x-axis

assert-x-axis body dir &optional id (x-label "x")(y-label "y"))
Description Check the correctness of the coordinate axes drawn by the student.
Example
  "(assert-x-axis 'b1 35 'axis-1 \"x1\" \"y1\")"
Returns
An entry status return value
Arguments
body:
the label given to the body the axis applies to
dir:
angle of the X axis from horizontal: 0dir360
id:
id assigned to the object by the workbench
x-label,y-label:
the labels given to the axes by the student (defaults to "x" and "y" if not given)
Side Effects
Adds X and Y axes to (student-entries). Asserts axes observed to Assessor.
