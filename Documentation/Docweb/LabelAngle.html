label-angle

label-angle label degrees id-vector1 id-vector2 id-angle &optional axis
Description
Assigns the given label to the angle between two objects with given degrees size.
Example
   
"(label-angle \"$$q\" 60 'Force-21 'Axis-32 'Angle-44 'posy)"
Returns
An Entry Status Return Value.
Arguments
label
The label given the angle by the student, a string
degrees
the size of the angle
id-vector1,id-vector2
the ids of the two vectors (or vector and axis) forming the angle. Either (but not both) of the ids may refer to an axis.
id-angle
the id of the new angle object
axis
the part of the coordinate system that is used (posx, negx, posy, negy)
Side Effects
Adds angle label to the list of variables.
