Natural Language Generation

(as of 3/16/01)

This module allows the hint-formulating modules to more easily generate text strings that refer to objects.  The basic idea is to pass in an operator, proposition or term from the KB, and an atom describing what kind of phrase is needed (e.g., gerund, definite noun-phrase, indefinite noun-phrase, etc.), and this module returns the appropriate text.

The KB has macros of the form (defexpr <expr> ...) where <expr> unifies with the kinds of expressions passed into this module.  The remainder of the macro specifies the kinds of English required.
