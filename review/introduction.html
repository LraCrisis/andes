• Define scalar variables using the Text Tool.
  • When using text to define a variable, possible forms include:
    • "m is the mass of the ball"
    • "m: mass of the ball"
    • "m=mass of the ball"
  • Define vector variables using the text box of the Vector Tool. When you define a vector, variables for the magnitude, components, and angle are defined automatically. So, for vector F, the magnitude is F, components F_x F_y, and angle \thetaF are defined.
  • Define coordinate system before vectors using the Axes Tool.
  • Write equations using the Equation Tool; all variables used must be previously defined.
  • Double click on a text or equation box to make a correction, then reenter.
  • Hit enter after entering any information in a text box.
  • To find the numerical value of a quantity:
    • Enter equations and value assignments required to solve for variable V.
    • In an equation box enter V=?
  • Enter the final answer in a text box in the form like answer: 1.23 m/s or answer: v1=1.23 m/s
  • If you are stuck or confused, hit the help button, "?" at the lower right corner of the help pane.
  • If you have a comment or complaint, simply type it into the text box at the bottom of the help pane. These are logged and will be read by us later.

Known Issues

There are plenty of known problems, including:
  • There is no method for handling z-axis vectors.
  • Multiple-choice and check boxes don't work.
  • Math notation (esp. Greek letters) is not handled.
  • The notation for entering Greek letters, like pi, is opaque.
