Drawing Tools

  • Define the coordinate system by using the Axis Tool to draw axes.  Click on the tool and drag out the x-axis at the desired angle.  The default axis labels x and y will be displayed; hit "Enter".
  • Define bodies (objects) with either the Ellipse or Rectangle Tool.   Click on the tool, drag out the shape, and define a body.
  • Define scalar variables using the Text Tool .   Click on the tool, drag out a box, and define a scalar variable.
  • Define vector variables using the Vector Tool .   Click on the tool, drag out a vector at the appropriate angle, if known, and define the vector.  Variables for the magnitude, components, and angle are defined automatically.  So, for vector F, the magnitude F, components F_x F_y, and angle \thetaF are defined.
  • Write equations using the Equation Tool .  Click on the tool, drag out an equation box, and write an equation.

Defining Quantities

When you draw an object, you can use the Text Box to define an associated quantity.  If you wish to define a variable, possible forms include:
  • m is the mass of the ball
  • m: mass of the ball
  • Let m be the mass of the ball
After defining a quantity, the text box will turn orange.  You can now place the box anywhere on the drawing area.

To make a correction, Double click on a text box, edit the text, then re-enter.

Entering equations

Equations are entered with the Equation Tool.
  • To find the numerical value of a quantity:
    • Enter all equations required to solve for variable V.
    • In an equation box enter:  V=?
  • Enter the final answer in a text box in the form like answer: 1.23 m/s or answer: v1=1.23 m/s
  • All variables used must be previously defined.
