Tool bar

  • Define the coordinate system by using the Axis Tool to draw axes.
  • Define bodies with either the Ellipse or Rectangle Tool.
  • Define scalar variables using the Text Tool .
  • Define vector variables using the Vector Tool .  When you define a vector, variables for the magnitude, components, and angle are defined automatically.  So, for vector F, the magnitude F, components F_x F_y, and angle \thetaF are defined.
  • Write equations using the Equation Tool ; all variables used must be previously defined.
  • When defining a variable, possible forms include:
    • m is the mass of the ball
    • m: mass of the ball
    • Let m be the mass of the ball
  • After defining a quantity, the text box will turn orange.  You can now place the box anywhere on the drawing area.
  • Double click on a text or equation box to make a correction, then re-enter.
  • Hit enter after entering any information in a text box.
  • To find the numerical value of a quantity:
    • Enter all equations required to solve for variable V.
    • In an equation box enter:  V=?
  • Enter the final answer in a text box in the form like answer: 1.23 m/s or answer: v1=1.23 m/s
  • If you are stuck or confused, hit the help button at the lower right corner of the Tutor pane.
  • If you have a comment or complaint, type it in the text box at the bottom of the Tutor pane.  These are logged and will be read by us later.
