Drawing Tools

  • Define the coordinate system by using the Axis Tool to draw axes.  Click on the tool and drag out the x-axis at the desired angle.  The default axis labels x and y will be displayed; hit "Enter".
  • Define bodies (objects) with either the Ellipse or Rectangle Tool.   Click on the tool, drag out the shape, and define a body.
  • Define scalar variables using the Text Tool .   Click on the tool, drag out a box, and define a scalar variable.
  • Define vector variables using the Vector Tool .   Click on the tool, drag out a vector at the appropriate angle, if known, and define the vector.  Variables for the magnitude, components, and angle are defined automatically.  So, for vector F, the magnitude F, components F_x F_y, and angle θF (type \thetaF) are defined.
  • Write equations using the Equation Tool .  Click on the tool, drag out an equation box, and write an equation.
  • Define lines and rays using the Line Tool .   Click on the tool, drag out a line in the appropriate direction, if known, and define the line.  Variables for the length and angle are defined automatically.  So, for line R, the length R and angle θR (type \thetaR) are defined.

Math notation

If you want to use a Greek letter, type a backslash, \, and the name of the letter. For instance, if you type \alpha, the Greek letter α will appear when you enter the equation or definition.

Subscripts are written using the underscore, like F_x, and exponents are expressed with the carat, x^2.  Scientific notation for numbers can be written as 3.06E-6 or as 3.06*10^-6.

Defining Quantities

When you draw an object, you can use the text entry box to define that object.  (Or use the Text Tool to define a scalar quantity.)  If you wish to define a variable for that object or quantity, possible forms include:
  • m is the mass of the ball
  • m: mass of the ball
  • Let m be the mass of the ball
After defining a quantity, the text box will turn orange.  You can now place the box anywhere on the drawing area.  To make a correction, click twice on the text box, edit the text, then re-enter.

Andes does not understand possessive apostrophes, for example:  the ball's velocity.  Rather, use something like:  the velocity of the ball.

You can create an object that is not evaluated by the Andes help system by entering text that starts with punctuation character like:  . , ? ! # % : ;.

Entering equations

Equations are entered with the Equation Tool.
  • To find the numerical value of a quantity:
    • Enter all equations required to solve for variable V.
    • In an equation box enter:  V=?
  • Enter the final answer in a text box in the form like answer: 1.23 m/s or answer: v1=1.23 m/s.  Andes expects three siginificant digits of precision for final answers.
  • All variables used must be previously defined.

Getting Help

  • If you are stuck or confused, click on the hint button at the bottom of the Tutor pane.
  • If you have a comment or complaint, type it in the text box at the bottom of the Tutor pane.  These are logged and will be read by us later.

Editing

  • To change some text, click twice on a text box.
  • To delete an object, click on the object to select it and then hit the delete or backspace button to delete it.
